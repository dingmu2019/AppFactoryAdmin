# Jobs Experience Booster

这个 Skill 旨在将乔布斯的“简约”、“整合”和“惊喜”理念注入 AdminSys 项目。

## 乔布斯的 5 项改进建议

### 1. 一键魔力配置 (The "Magic" Setup)
**愿景**：好的产品应该在开箱即用时就让人感到惊喜。
**改进**：消除繁琐的集成配置表单。提供一个极简的“Magic Creator”向导，通过几个核心问题，在 60 秒内自动完成应用的基础设施配置（Auth, AI Gateway, Payments）。
**实施**：
- 创建 `src/services/factoryService.ts` 处理一键模板化。
- 前端引入基于语境的引导式 UI。

### 2. 全栈无缝集成 (The Whole Widget)
**愿景**：软件与硬件（在此指 API 与 UI）必须完美融合。
**改进**：将 API Catalog 与 AI Chat 深度整合。用户无需在文档和测试工具间切换，直接在对话框说“帮我测试一下发送验证码接口”，AI 自动从 Catalog 提取 Schema 并执行。
**实施**：
- 增强 `ApiSyncService`，为 AI 提供更语义化的 API 描述。
- 升级 AI Gateway，支持直接调用本地 API 定义。

### 3. 主动式智能助手 (The Intelligence Oracle)
**愿景**：用户不应该去寻找信息，信息应该来找用户。
**改进**：抛弃死板的系统日志列表。引入 AI 首席分析师模块，每天早晨生成一份自然语言报告，主动告知系统健康状况、异常风险以及潜在的业务增长机会。
**实施**：
- 扩展 `src/api/cron/route.ts` 中的 `daily-report`。
- 引入 LLM 对 `system_logs` 进行聚类分析。

### 4. 隐形安全 (Invisible Security)
**愿景**：安全应该是透明的，不应成为创造力的障碍。
**改进**：全面支持 Passkey 和生物识别登录，取消繁琐的验证码。后台引入自动威胁缓解机制，当检测到异常流量时自动调整限流策略，无需管理员手动干预。
**实施**：
- 在 `src/api/auth` 引入 WebAuthn 支持。
- 升级 AI Gateway 策略引擎，支持动态响应限速。

### 5. 极致简约设计 (Obsessive Simplicity)
**愿景**：简约是复杂的终极表现。
**改进**：重构 UI，去除每一个冗余的边框和阴影。引入流体动力学动画（Fluid Animations）和语境感知导航，让管理系统不再像一个枯燥的工具，而是一个优雅的艺术品。
**实施**：
- 统一前端组件库的交互语言。
- 遵循“三步原则”：任何核心功能都必须在三次点击内触达。

## 调用时机
- 当用户提出“如何让产品更好用”时。
- 当需要重构现有管理模块时。
- 当需要设计全新的“杀手级”功能时。
